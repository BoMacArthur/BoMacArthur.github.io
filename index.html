Google Earth Engine Light Code Style

This example demonstrates Google Earth Engine code syntax styling with a light theme:

    
      // This is a comment (green)
      var dataset = ee.ImageCollection("MODIS/006/MOD13Q1");
      var ndvi = dataset.select("NDVI"); // Keyword (blue)
      var visParams = {min: 0.0, max: 9000.0}; // Variable (dark blue)
      Map.addLayer(ndvi, visParams, "NDVI"); // Function (brownish-yellow)
      print("NDVI Layer Added", ndvi); // String (red)
